# MyReads Project

MyReads is an app that allows you to look up and catalog books that you _have read_, _want to read_ or are _currently reading_.


## Installation

```
$ git clone https://github.com/ahmedbaligh7/myReadsApp.git
$ cd myReadsApp
$ npm install
$ npm start
```
